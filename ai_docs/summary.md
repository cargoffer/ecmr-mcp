# ECMR MCP Summary

Model Context Protocol server connecting AI agents to Cargoffer ECMR API (electronic consignment notes).

## Use: Enable AI agents to manage transport logistics

- eCMR lifecycle: create, read, update, delete, lock
- Driver & vehicle management
- Digital signatures (sender, pickup, delivery)
- QR code generation/validation

## MCP: 20+ tools exposed as JSON-RPC 2.0

## API: ecmr.api.cargoffer.com

## Keywords: mcp, ecmr, ai agents, transport, logistics, digital freight
