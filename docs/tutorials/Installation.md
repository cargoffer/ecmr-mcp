# Installation

## Requirements

- Node.js 18+
- npm or yarn

## Installing the MCP Server

```bash
npm install @cargoffer/ecmr-mcp
```

## Configuration

Set your ECMR API key:

```bash
export ECMR_API_KEY="your_api_key_here"
```

## Running

```bash
npx @cargoffer/ecmr-mcp
```

For more details, see the [README](https://github.com/cargoffer/ecmr-mcp).
